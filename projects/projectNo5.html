Website Testing Object Recognition Using YOLOv8

Implemented a web-based application that uses the YOLOv8 object recognition algorithm to test the accuracy and speed of object detection on various datasets. This project focuses on both object segmentation and detection. It allows users to identify and outline objects within images and videos in real-time using advanced AI models like YOLOv8, all within a simple and intuitive interface powered by Streamlit.
